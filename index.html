[25-04-2025] Version 2.0.33.1 Released

Last week, we successfully deployed version v2.0.33.1 of the CRVS system on Thursday at 05:00 AM. The update included several significant enhancements and new features aimed at improving the system's functionality and security. Additionally, a hotfix v2.0.33.3 was released to address some critical issues identified post-deployment.

Updates in v2.0.33.1:

  • 1. QR Code integration:

    • Added QR codes to the Birth Certificate PDFs to enhance verification and accessibility.

  • 2. Language and Style Updates:

    • Updated the PDF templates to include the Kurdish language.
    • Improved the overall style and layout of the PDFs for better readability and presentation.

  • 3. Security Enhancements:

    • Implemented a new JWT Authentication system to secure the application and facilitate seamless integration with NDC Systems.

  • 4. Performance Optimizations:

    • Optimized database queries and API responses to reduce loading times and improve overall system responsiveness.

  • 5. User Interface (UI) Improvements:

    • Redesigned several key pages for a more intuitive and user-friendly experience.

Hotfix v2.0.33.3 Details:

  • 1. Critical Bug Fixes:

    • Resolved an issue causing occasional data corruption during bulk record updates.
    • Fixed a critical security vulnerability related to cross-site scripting (XSS) in user input fields.

  • 2. Stability Improvements:

    • Enhanced system stability under high load conditions by optimizing resource management.
